# update_chronicler
A script for Debian-based systems to semi-automatically update Chronicler.

## Instructions for use
1. Download `update_chronicler.sh` and `update_chronicler.py` and store them in the same directory.
2. Make sure `update_chronicler.sh` has Execute permissions.
3. Open a terminal and navigate to the directory the downloaded files were placed in.
4. run `.\update_chronicler.sh`
